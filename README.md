# Nike-Sales-Dashboard
Designed a 3-page Power BI dashboard designed to analyze Nike shoe sales performance, product information, customer feedback, and regional sales trends. The dashboard uses synchronized slicers to provide a seamless analytical experience across all pages.

## Problem Statement
Nike product data includes sales, ratings, reviews, pricing, and regional performance information spread across multiple records. The goal was to build an interactive dashboard that helps users understand product performance and sales trends efficiently.

## Tools Used
- Power BI
- DAX
- Excel
- Power Query

## Key KPIs Shown 
- Overview & Insight Pages: Total Sales Price, Total Listing Price, Overall Rating, Number of Reviews.
- Performance Page: Total Unts Sold, Current Year Units Sold, Previous Year Units Sold.

## Dashboard Features 
- Overview Page: Shoe name and description, Product images from multiple angles, Button slicer for shoe selection.
- Insight Page: Detailed product information, Sales and listing price metrics, Ratings and review analysis, Five-angle product image display.
- Performance Page: KPI cards for sales performance, Region-wise units sold analysis, Month-wise units sold trends, Product image visualization, Sync slicer affecting all dashboard pages.

## Insights Generated
- Identified best-performing shoe models.
- Compared current and previous year sales.
- Analyzed customer ratings and reviews.
- Determined top-performing sales regions.
- Tracked monthly sales trends and seasonality.
